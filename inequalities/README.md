These are a list of inequalities that I have learnt and used to solve different types of inequailty problems.
This is probably not a complete list of all the possible inequalities that might be used but definetely covers all the main and  the ones most likely to be used.
For additional inequalities that may have a possibility of being used to solve a problem...check the link in the pdf but I belive that these cover the vast majority of them.

You will see your more standard inequalities like AM-GM, Cauchy-Schwarz, Jensen which can be stated and used without needing to be proved but there are other inequalities as well that are probably less standard like Minkowski's which may need to be proven in the contest when you write your solution which means that it is all the more important to know how to prove each of them.
